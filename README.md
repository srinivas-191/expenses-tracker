# Expense Tracker

> A small React + Vite app for tracking personal expenses.

## Features
- Add and list expenses
- Simple, responsive UI built with Bootstrap
- Test coverage with Vitest

## Tech stack
- React
- Vite
- Bootstrap
- Vitest for tests

## Requirements
- Node.js 18+ and npm or yarn

## Setup
Clone the repo and install dependencies:

```bash
git clone https://github.com/srinivas-191/expenses-tracker.git
cd expenses-tracker
npm install
```

## Available scripts

- Start dev server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```

- Preview production build:

```bash
npm run preview
```

- Run tests:

```bash
npm run test
```
